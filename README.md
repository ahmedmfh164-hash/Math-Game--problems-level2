# Math-Game--problems-level2

🧮 Math Quiz Game (C++)

A simple console-based Math Quiz Game built using C++.

This program generates random math questions and tests the user's skills in solving them.

---

🚀 Features

- 🎲 Random question generation
- ➕ Multiple operations:
  - Addition
  - Subtraction
  - Multiplication
  - Division
  - Mixed mode
- 🎯 Difficulty levels:
  - Easy
  - Medium
  - Hard
  - Mixed
- 📊 Score tracking (Right / Wrong answers)
- ✅ Pass / Fail result

---

🛠️ Technologies Used

- C++
- Standard Library ("cstdlib", "ctime", "iostream")
- Console-based UI

---

▶️ How to Run

1. Compile the program:

g++ main.cpp -o math_quiz

2. Run:

./math_quiz

---

🎮 How It Works

1. Enter number of questions.
2. The system generates random math problems.
3. You answer each question.
4. At the end:
   - Shows total correct answers
   - Shows total wrong answers
   - Displays pass/fail result

⚠️ Notes

- Division by zero is handled safely.
- Questions are generated randomly every time.
- Uses integer math only.
